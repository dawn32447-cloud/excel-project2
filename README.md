Key Performance Indicators (KPIs)


The dashboard tracks several critical business metrics to evaluate Swiggy's sales performance:

Total Sales: The overall revenue generated from orders.

Total Orders: The count of individual orders placed.

Average Rating: The mean customer rating across different food categories and quarters.

Sales by Food Type: A breakdown of revenue between Veg and Non-Veg items.

Daily & Weekly Sales Trends: Identification of peak ordering days (e.g., Saturday and Sunday) and weekly fluctuations.

Geographic Performance: Sales distribution across different States and Top 5 Cities (e.g., Bangalore as a top performer).

Dashboard Features

The project incorporates advanced Excel functionalities to create an interactive user experience:

Interactive Slicers: Dynamic filters for Months, Restaurant Names, and Categories that update all charts simultaneously.

Custom Map Visualization: A geographic heat map showing sales density across Indian states.

Dynamic Donut & Bar Charts: Visual representations of food type distributions and sales trends with custom gradients.

Linked Picture KPIs: Using the "Linked Picture" tool to display formatted table data (like quarterly performance) directly on the dashboard.

Navigation Buttons: Functional buttons to switch between the Dashboard, Analysis, and Raw Data sheets .

Automated Data Cleaning: Formula-driven columns to categorize data (e.g., extracting "Food Type" from dish names).
